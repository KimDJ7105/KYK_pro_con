규칙 정리본

(게임 진행)

[1라운드] : 1~25의 숫자 중 플레이어들은 임의의 숫자를 선택하고, 해당 위치에 키카드를 배치함. 총 3개의 키카드가 배치되며, 최우선 목표는 해당 키카드를 전부 회수하는 것임.

1.1. 최대 4인이 지정된 위치(본 게임과 동일)에서 스폰하며, 1인당 3개의 말을 가지고 플레이 가능.

1.2. 각자의 턴이 돌아올 때마다 10의 행동력이 주어지며, 플레이어들은 행동력을 소모하여 이동, 수집, 공격, 카드 사용 등의 행동이 가능함. 각 말은 총 5의 체력이 주어지며, 이 이상으로 회복시킬 수 없고 체력이 0이 될 경우 해당 말은 비활성화되어 뒤집은 상태로 해당 구역에 배치함.

1.3. 키카드를 획득한 플레이어는 자신의 턴에 7의 행동력과 키카드를 소모하여 단 1번 통로 봉쇄, 함정 설치(함정이 설치된 구역을 지나간 플레이어는 카드 덱에서 함정 카드가 나올때까지 카드를 뽑고, 뽑은 함정 카드의 효과를 적용받은 뒤 뽑았던 카드를 전부 덱으로 되돌리고 셔플함.) 같은 특수 기능을 사용할 수 있으며, 3장의 키카드가 모두 회수되었을 경우 1라운드가 종료되며, 2라운드가 시작됨.

[2라운드] : 모든 플레이어가 차례를 마칠 때마다 맵의 외곽 지역을 순회하는 분쇄기가 맵에 배치되며, 랜덤한 구역에 오브젝트가 배치됨. 최우선 목표는 해당 오브젝트를 회수하는 것임.

2.1. 2라운드가 시작됨과 동시에 맵 외곽 구역에 분쇄기가 배치되며, 분쇄기는 모든 플레이어들의 차례가 종료될 때마다 맵 외곽을 직진으로 이동하여 경로상에 존재하는 모든 말을 즉사시킴.

2.2. 오브젝트를 획득할 경우 획득한 플레이어는 자신의 턴부터 세어서 총 (플레이어 수 + 1)턴을 소유하고 있어야 하며, 버티는 도중 오브젝트를 획득한 말의 체력이 0이 되었을 경우 해당 말을 사살한 플레이어의 말에게 오브젝트의 소유권이 넘어감. 소유권이 넘어가도 소유한 턴 수는 누적되기 때문에 나중에 오브젝트를 얻은 플레이어일 수록 버텨야 하는 시간은 짧아지게 됨. 

2.3. 만약 소유 기한을 전부 채웠다면, 무작위 구역에 탈출구가 열림. 오브젝트를 가진 말이 해당 지역으로 탈출에 성공하면 해당 말을 가진 플레이어의 승리임. 

--------------------------------------------------------------------

지원 카드 -> 체력 회복, 추가 행동력 제공, 보급품 투하(행동력 지불 후 카드 2장 제공) 등

공격 카드 -> 샷건(사거리 1칸, 3데미지), 라이플(사거리 2칸, 1데미지, 동종 카드가 여럿 있을 경우 2회까지 연속으로 내기 가능), 근접무기(동일 구역 내에서만 사용 가능, 1~6의 눈금을 가진 주사위를 던져 3 이상의 수가 나올 경우 4의 데미지를 줄 수 있지만 2 이하의 수가 나올 경우 반격 여부와 상관 없이 공격자가 역으로 2 데미지를 입음)

함정 카드 -> 획득과 동시에 자동으로 사용되며, 획득한 플레이어에게 일정 페널티를 부과함.(예 : 무기/지원 카드 2개 제거, 잔여 행동력 3 차감, 보유 중인 모든 말에게 1 데미지 등)

덱에 들어가는 카드 비율 = 2(지원) : 1(공격) : 1(함정), 총 80장이 덱에 들어감.

--------------------------------------------------------------------
이동 : 행동력 1을 차감하는 것으로 연결된 구역이나 통로로 이동 가능. 이동에 따른 행동력 소모는 말마다 개별적으로 적용됨.

환풍구 이동 : 보라색으로 칠해진 각 맵의 모서리(버텍스)와 정 중앙의 구역에서 행동력 3을 차감하는 것으로 다른 보라색으로 칠해진 구역으로 이동 가능, 환풍구를 사용한 말은 다음 자신의 턴까지 이동이 불가능함.(단, 이동을 제외한 다른 행동은 가능함) 

수집 : 행동력 1을 차감하는 것으로 현재 있는 구역에서 카드 혹은 비활성화된 아군/적군, 퀘스트 아이템 획득. 획득 가능한 카드는 3종류로, 지원 카드와 공격 카드, 함정 카드로 나뉨. 한 턴에 한 구역에서 획득 가능한 카드는 해당 지역에 놓여진 아군 말의 수와 상관 없이 최대 1장이며, 그 이상으로 수집을 사용 시에는 아군/적군 회수 or 퀘스트 아이템 획득만 가능함. 

공격 : 행동력 2를 차감 + 공격 카드를 사용하는 것으로 카드에 적힌 사거리 내에 존재하는 말을 지정하여 공격 가능, 피격당한 말을 보유한 플레이어는 자신이 보유한 공격 카드 사거리 내에 공격한 말이 존재할 경우 해당 카드를 소진하여 반격 가능함.

카드 사용 : 카드를 버리고 해당 카드에 적힌 효과를 발동(대가 지불이 필요한 경우 지불 후 발동), 자신의 턴일 경우 언제든지 사용 가능.

부활 : 플레이어가 게임을 시작한 시작 지점에서 행동력 5를 차감하는 것으로 비활성화된 아군 말을 활성화 시킬 수 있음.
